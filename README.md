# Pointex

A strategic grid-based multiplayer.

---

## 🎮 About the game

Pointex is a strategic grid game inspired by classic paper games played in school notebooks.

Two players (Blue 🔵 and Red 🔴) take turns placing points on a grid.  
When a player completes a square, they score a point.

The game ends when all points are filled, and the player with the highest score wins.

---

## 🚀 Features

- Turn-based gameplay (Blue vs Red)
- Score tracking system
- Square detection logic
- Game Over screen with winner display
- Replay system
- Responsive grid-based UI
- Animated feedback (Replay messages)

---

## 🛠 Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- React Icons

---

## 🎯 Gameplay Rules

- Players alternate turns
- A point is placed on grid intersections
- When 4 connected points form a square, the player scores
- The square is marked with a number
- Game ends when all intersections are filled
